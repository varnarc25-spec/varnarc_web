# 02 — Tech Stack

Locked choices for Varnarc Platform. Do not change without updating `01-Architecture.md` and `AI_RULES.md`.

| Layer | Technology |
|-------|------------|
| Public / Admin | Next.js 15 App Router |
| API | NestJS |
| ORM | Prisma |
| Database | Neon PostgreSQL |
| Auth | Auth0 |
| UI | Tailwind CSS + shadcn/ui patterns via `@varnarc/ui` |
| Client data | TanStack Query |
| Storage | Cloudinary |
| Email | Resend |
| Deploy | Docker + Google Cloud Run |
| CI/CD | GitHub Actions |
| Monorepo | pnpm + Turborepo |
